TwitchTweaks

Enhance Your Twitch Experience

A powerful browser extension for customizable features and utilities on Twitch.

Get Started View on GitHub

Features

File Uploader

Drag & drop or paste files directly into chat. Automatically uploads to litterbox and copies the link.

Media Previews

Preview media files directly in twitch with copy and open options.

Text Previews

Preview text files directly in twitch with copy and open options.

Osu! Previews

Preview osu! beatmap links to see detailed stats, cover art, and download options.

Updater

The extension will automatically detect the update and prompt you to update.

More soon...

Stay tuned for even more exciting features coming to TwitchTweaks.

Installation

1

Clone the repository: git clone https://github.com/ikuza47/TwitchTweaks.git

2

Open your browser's extension page (e.g., chrome://extensions).

3

Enable "Developer mode" and click "Load unpacked".

4

Select the cloned TwitchTweaks folder.
